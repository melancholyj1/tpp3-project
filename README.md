# Zenly Clone 🌍

Социальная карта с отслеживанием друзей в реальном времени.

## Особенности
- 📍 Геолокация в реальном времени через Socket.IO.
- 🗺️ Интерактивная карта (Leaflet).
- 🔍 Поиск мест (POI) через Foursquare API.
- 🔐 Авторизация на JWT.
- 🛡️ Защита сервера (Helmet, Rate Limit).

## Как запустить проект

### 1. Требования
- **Node.js** (версия 18 или выше)
- **Python 3** (для использования скрипта автоматизации)

### 2. Настройка окружения
1. Склонируйте репозиторий.
2. Создайте файл `.env.local` в корне проекта.
3. Скопируйте содержимое из `.env.example` в `.env.local` и подставьте свои ключи:
   - **MONGODB_URI**: Ваша строка подключения к MongoDB Atlas.
   - **FOURSQUARE_API_KEY**: API ключ от Foursquare (для поиска мест).
   - **JWT_SECRET**: Любая секретная фраза.

### 3. Запуск
Самый простой способ — использовать автоматизацию:
```bash
python run.py
```
Скрипт сам проверит наличие конфигурации, установит зависимости (`npm install`) и запустит сервер разработчика.
